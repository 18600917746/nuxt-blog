# nuxt-blog

## 基于Nuxt.js服务器渲染(SSR)构建的个人博客系统

### 线上Demo地址: [Brian's Blog](http://www.brianlee.cn)

## 环境规范：

> Vue.js >= 2.0

> Nuxt.js >= 1.0.0

> Node.js > 8.0

> MongoDB >= v3.6.3

## 目录说明：

* `nuxt/`:**前端UI界面**

* `server/`:**后端服务功能**

* `serviceUI/`:**后端UI界面源码**

## 构建与运行

> 构建与运行及详情在每个目录下的README说明

```shell

# nuxt localhost:8081

$ npm install

# server localhost:3000

$ npm install

# serviceUI localhost:8082

$ npm install

```
